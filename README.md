# Machine Learning Codes 
> A comprehensive collection of Machine Learning algorithms and experiments.

## 📌 Project Overview
This repository contains Python implementations of various machine learning models, ranging from fundamental statistical methods to advanced neural networks. Each directory includes the source code, dataset (or links), and a brief explanation of the results.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / Google Colab / VS Code

## 🧪 Experiments Covered
1.  **Linear & Logistic Regression:** Implementing gradient descent and cost functions.
2.  **Decision Trees & Random Forests:** Classification tasks using `Information Gain` and `Gini Index`.
3.  **Support Vector Machines (SVM):** Exploring different kernels (Linear, RBF, Polynomial).
4.  **K-Nearest Neighbors (KNN):** Feature scaling and finding the optimal k value.
5.  **K-Means Clustering:** Unsupervised learning and the Elbow Method.
6.  **Neural Networks:** Basic Perceptron and Multi-Layer Perceptron (MLP) implementations.
7.  **PCA (Principal Component Analysis):** Dimensionality reduction for visualization.

## 🚀 How to Use
1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/your-username/ML-Lab-Records.git](https://github.com/your-username/ML-Lab-Records.git)
    ```
2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run Notebooks:** Open the `.ipynb` files in Jupyter or Colab to view the step-by-step execution.
